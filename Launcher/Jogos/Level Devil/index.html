Levil Devil

Como jogar

Use ← → ou A/D para mover. ↑ ou Espaço para pular. Evite espinhos e colete estrelas.

Objetivos

  • Colete todas as estrelas douradas (cada uma vale 10 pontos).
  • Evite tocar os espinhos — eles matam instantaneamente.
  • Chegue até a bandeira para completar o nível.
  • Use o ambiente com cuidado — plataformas são estreitas e alguns espinhos se movem.
